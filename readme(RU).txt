Эта программа может быть использована ресторанами для
контроля за статусом столов: свободны ли они или нет.
Кроме того, с помощью этой программы вы можете резервировать столы на какое-нибудь время.

В программе предусмотрены исключения:

1.Если запрос пустой
2.Если не заданы параметры для команд



Как использовать программу:
В этой программе существуют такие команды:
1.List
 Функция: открывает список всех столов
 Пример: List
2.Edit
 Функция: изменяет статус выбранного стола
 Пример: Edit номер_стола новый_статус
	 Edit 1 Занят (где '1' - номер стола, а 'Занят' - новый статус стола)
3.Info
 Функция: выводит информацию о выбранном столе
 Пример: Info номер_стола
	 Info 2 (где '2' - номер стола)
4.Reserve
 Функция: резервирует стол на определённое время
 Пример: Reserve номер_стола время_резервирования
	 Reserve 10 16:00 (где '10' - номер стола, а '16:00' - это время резервирования)
